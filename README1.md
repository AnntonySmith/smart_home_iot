# Умный дом – IoT система

Проект "Умный дом" с веб-интерфейсом на Flask, автоматическим логированием и аналитикой.

## Запуск

1. Скачайте проект или клонируйте: `git clone https://github.com/ваш_логин/smart_home_iot.git`
2. Установите Python, создайте виртуальное окружение, установите Flask: `pip install flask`
3. Запустите: `python app.py`
4. Откройте http://127.0.0.1:5000
